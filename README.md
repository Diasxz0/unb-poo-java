UnB - Programação Orientada a Objetos (Java)
Este repositório contém os exercícios e projetos desenvolvidos por Warlley Dias de Medeiros durante o 3º semestre da graduação em Engenharia na Universidade de Brasília (UnB/FCTE). O foco principal é o aprendizado de Java, Lógica de Programação e os pilares da Programação Orientada a Objetos (POO).

📂 Estrutura do Repositório
O projeto está organizado por níveis de complexidade para facilitar a revisão dos conceitos:

1. 01-logica-e-fundamentos
Contém a base da sintaxe Java e estruturas essenciais:

Fundamentos: Manipulação de variáveis, operadores de incremento/decremento e entrada de dados com Scanner.

Estruturas de Controle: Exemplos práticos de if/else, switch-case, e laços de repetição (for, while, do-while).

Exercícios Base: Pequenos desafios lógicos, como calculadoras simples e jogos de adivinhação.

2. 02-poo-basica
Focado nos conceitos fundamentais da orientação a objetos:

Classes e Objetos: Estruturação de modelos e instanciamento.

Relacionamentos: Exemplos de Agregação e Composição (ex: sistema de contatos com endereço e telefone).

Pilares POO: Implementações de Herança e Interfaces utilizando exemplos práticos.

3. 03-topicos-avancados
Projetos que utilizam recursos mais complexos da linguagem:

Tratamento de Exceções: Criação e gestão de exceções personalizadas (ex: AgendaCheiaException).

Mini-Projetos: Sistemas mais completos, como o Gerenciador de Agenda, que integram lógica de negócios e persistência em memória.

🛠️ Tecnologias Utilizadas
Linguagem: Java (OpenJDK).

IDE: Eclipse / VS Code.

Controle de Versão: Git & GitHub.
